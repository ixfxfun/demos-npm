keyboard-remote

This sketch demonstrates sending keyboard events from one device/sketch to another. For example, have a sketch running on a mobile be controlled from your laptop.

By default, the sending will only work on your own machine. Try this out by opening the sender and receiver sketches in different windows.

Multiple senders and receivers can be used.

  • sender
  • receiver

Please see README.md and ixfx guide for more details on connecting between devices.
